邮件书写6要素
```
* 标题
* 收件人/上款
* 时间点
* 抄送
* 突出重点
* 当面或电话沟通过，邮件中再次确认
```
邮件标题
```
* 长度够长，16字以内
* 标题符合邮件意图
* 标题可以添加称呼，引起收件人注意
```
邮件收件人
```
* 考虑清楚收件人有哪些
* 明白写好上款，以便对方意识到“自己有细看这个邮件的责任”
* 上款宜直接称呼对方的名字，人较多时，宜称呼该组织的名字
* 一个邮件中给多人分别交代事情，应分成多段，每段分别写明上款
```
时间点
```
* 给对方提出需求的，应该给出明确希望的时间点，风格应明确但不生硬（例如“这件工作需要在15日下班前完成，如果你觉得这个时间有问题，请今天回信或来电话我们商量”）
* 对方必出需求时，没有写明时间点、紧急程度等问题，自己应该在沟通了时间点和紧急程度之后，回信给出相应的信息
* 跟踪bug/case的回复邮件，必须写明：
    * 问题
    * 分析
    * 结论
    * 计划
    * 时间点
```
抄送
```
* 跨部门、跨组的合作事宜，要抄送双方直接经理
* 在拿不准是否应该抄送经理时，一律抄送
* 同理，如果其他部门的人给你写的邮件没有抄送经理，需要打回，请求再次发送并抄送给双方的经理
```
突出重点
```
* 时间点
* 人名
* 不同收件人
* 邮件要简洁，避免乱标重点
```
进阶
```
* 注意语法，避免错误
* 邮件布局合理、段落合理、适当修辞、言语生动
* 注意review，保证邮件质量
```
