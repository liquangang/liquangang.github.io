### 常用命令
#### 常用流程
* git pull
* git add .; git add [filename]
* git commit -m ""; git commit --amend
* git push
* git status: 查看本次修改

#### stash
* git stash
* git stash pop
* git stash list
* git stash clear
* git stash drop [stashId]: 删除某个暂存
* git stash apply [stashId]: 使用某个暂存

#### commit
* git reset --soft 想撤回到的commit：撤回commit
* git log: 查看commit 记录

#### dif
* git diff
