Java-String

String

  • 特点：
    • 效果上相当于字符数组（char[]），实际上底层原理是字节数组（byte[]）

构造方法

  • 不同构造方法特点
    • 使用””方法创建的对象实际指向的是同一个内存地址
    • 使用new创建的对象即时字符换内容相同，对象指向的内存地址也不同

字符串比较

  • ==（实际都是比较变量保存的值是否相同）
    • 基本数据类型是比较值
    • 引用类型比较的是地址值

StringBuilder

  • 特点：
    • 去掉字符串拼接过程中构造新对象的行为（解决浪费内存空间且耗时的问题）
    • 内容可变
