关键字、标识符

关键字

  • 特点：
    • 小写

标识符

  • 作用：用于给常量、变量、函数、语句块等进行命名
  • 组成:
    • 字母
    • 数字
    • 下划线（_）
    • 美元符号（$）
  • 特点：
    • 第一个字符不能是数字
    • 不能使用关键字和保留字
    • 无长度限制
    • 区分大小写

final

  • 自我理解：即被修饰的东西不能被改变
  • 特点：
    • 变量值不能变，但是引用类型其属性值可以变
    • 类不能被继承
    • 方法不能被重写

static

  • 特点：可以被类的所有对象访问，包含类对象和普通对象
