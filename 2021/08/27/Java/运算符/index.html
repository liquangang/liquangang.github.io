运算符

概念

  • 运算符：
    • 在表达式中表示计算的符号
  • 表达式：
    • 由运算符、变量、常量等组成的有意义的组合

算数运算符

  • 组成：
    • +
    • -
    • *
    • \
    • %
  • 总结：
    • char运算时会转换为ASCII码的值
    • 多个类型运算时，结果为最大范围类型
    • 从左到右顺序执行
      • 例：1 + 2 + “abc” = “3abc”
      • 例：”abc” + 1 + 2 = “abc12”

赋值运算符

  • 组成：
    • =
  • 总结：
    • 可搭配其他运算符使用，例 +=、-=等，+=类似的赋值运算符内隐含强制类型转换

自增、自减运算符

  • 组成
    • ++
    • –
  • 总结：
    • 在参与复杂表达式中时，按顺序执行，++在前，先赋值，++在后，先+1，–同理

关系运算符

  • 组成：
    • ==
    • !=
    • >
    • >=
    • <
    • <=
  • 总结：
    • 结果为boolean

逻辑运算符

  • 组成：
    • &
    • |
    • !
    • ^
    • 短路逻辑运算符：
      • &&
      • ||
  • 总结：
    • 逻辑运算符是用来连接最终结果为boolean的表达式的
    • 逻辑运算符前后的表达式结果为boolean
    • 短路逻辑运算符有短路效果，如果第一个表达式就能确定最终结果，后面的表达式即不执行

三元运算符

  • 组成：
    • 例：a > b ? a : b;
